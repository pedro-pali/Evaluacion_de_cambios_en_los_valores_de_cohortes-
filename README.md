# Evaluacion_de_cambios_en_los_valores_de_cohortes-
Cambio en el comportamiento de cada cohorte con el tiempo (mes) en jupiter notebook donde se realizó la tabla pivote mostrando los cambios en las compras totales realizdas por los jugadores en cohortes mensuales y Python donde encontramos el número de compras y el número de jugadores que realizaron compras para cada cohorte y para cada mes.

## Herramientas:
- Python
- Pandas
- Microsoft Visual Studio Code

### Paso 1. Agrupación de cohortes
Se calcularon los cohortes siguiedo los pasos de calcular primero la fecha minima de cada compra, y agrupandolos por esta fecha.
